# NotchScript

NotchScript is an early Electron prototype for a JavaScript-driven macOS fluid notch.

## Run

```sh
npm install
npm run dev
```

## Prototype Controls

- `CommandOrControl + Shift + N`: toggle compact and expanded modes.
- `CommandOrControl + Shift + I`: toggle mouse interaction.
- The island starts in a compact pass-through state and briefly expands with a Hello World card.
